# 🎵 United Kingdom Top 50 Playlist Market Structure Analysis

## 📌 Project Overview

This project is an interactive Streamlit web application that analyzes the **United Kingdom Top 50 Spotify Playlist**. It provides insights into artist dominance, collaboration patterns, album distributions, explicit content, and track characteristics using data visualization and analytics.

The dashboard allows users to explore trends in the UK music market through interactive filters and charts.

---

## 🚀 Features

- 🏆 Artist Dominance Leaderboard
- 🤝 Collaboration Network Visualization
- 🎤 Solo vs Collaboration Analysis
- 🔞 Explicit vs Clean Content Analysis
- 💿 Album Type Distribution
- ⏱ Track Duration Analysis
- 📅 Date Range Filtering
- 🎯 Artist Filtering
- 🎵 Album Type Filtering
- 📊 Interactive Charts using Plotly
- 🌐 Streamlit Web Dashboard

## 📈 Dashboard Modules

### 1. Artist Dominance Leaderboard
Displays the most frequently appearing artists in the UK Top 50 playlist.

### 2. Collaboration Network
Visualizes collaborations between artists using a network graph.

### 3. Explicit vs Clean Analysis
Compares explicit and non-explicit tracks.

### 4. Album Type Distribution
Shows the distribution of Singles, Albums, and Compilations.

### 5. Track Duration Insights
Analyzes song duration using histograms and summary statistics.

### 6. Interactive Filters

Users can filter data by:

- Artist
- Date Range
- Album Type
- Solo vs Collaboration

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/UK-Top50-Playlist-Analysis.git
```

Move into the project folder

```bash
cd UK-Top50-Playlist-Analysis
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
